## Compiling

1. fetch the source code from github

```
git clone https://github.com/mlhoutel/garden
```

2. install the node dependancies

```
npm install
```

3. start a local development server

```
npm run start
```

4. build the website for release

```
npm run build
```

5. start a release preview

```
npm run preview
```
